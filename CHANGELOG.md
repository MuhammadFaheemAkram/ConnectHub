# Changelog

All notable changes to ConnectHub are documented here.

## 1.0.0 - 2026-06-16

### Added

- Project foundation with Kotlin, Compose, Material 3, Hilt, Navigation Compose, and reusable design-system components.
- Fake authentication with DataStore-backed session persistence.
- Offline-first feed with fake API, Room cache, likes, bookmarks, and pagination.
- Post detail, comments, create post, and comment persistence.
- Search, recent searches, bookmarks, profile, and edit-profile flows.
- Chat list, chat detail, simulated replies, notifications, and settings persistence.
- Room migrations through database version 4.
- Unit tests, Room DAO tests, Compose UI tests, and GitHub-ready documentation.
